# terminal-notes-with-git-temporal-database
Terminal Notes: Git-Powered Temporal Notes That Never Forget
Never lose a note again.
 Terminal Notes transforms Git into a temporal database for your 
knowledge, tracking every item with UUIDs across renames, moves, and 
deletions. Search across all time, resurrect deleted items with full 
context, and navigate instantly through infinite hierarchies.


# Clone the memory system
git clone https://github.com/yourusername/terminal-notes

# Enter your new digital hippocampus
cd terminal-notes

# Start remembering everything
python terminal_notes_ui.py

requirement : Linux, Python, vim/nvim, git
